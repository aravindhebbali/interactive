Introduction

Importing data is the first step in any data analysis workflow. If you cannot import data from differernt sources or file formats, you cannot proceed with data analysis. R allows you to import data from several sources and a variety of file formats. In this module, we will focus on

  • SAS
  • SPSS
  • STATA

haven

  • read_dta()
  • read_stata()
  • read_sav()
  • read_spss()
  • read_sas()

Other Libraries

  • SAS: sas7bdat (last updated in 2014; version 0.5)
  • STATA, SPSS: foreign (maintained; version 0.8)

STATA

haven::read_dta('airline.dta')

STATA

foreign::read.dta('airline.dta')

SPSS

read_sav('employee.sav')

SPSS

foreign::read.spss('employee.sav', use.value.labels = FALSE,
                   to.data.frame = TRUE)

SAS

read_sas('airline.sas7bdat')

SAS

sas7bdat::read.sas7bdat('airline.sas7bdat')
