Introduction

In this module, we will focus on the aesthetics i.e.Â color, shape, size, alpha, line type, line width etc. We can map these to variables or specify values for them. If we want to map the above to variables, we have to specify them within the aes() function. We will look at both methods.

Data

Introduction

ecom

Data Dictionary

  • id: row id
  • referrer: referrer website/search engine
  • os: operating system
  • browser: browser
  • device: device used to visit the website
  • n_pages: number of pages visited
  • duration: time spent on the website (in seconds)
  • repeat: frequency of visits
  • country: country of origin
  • purchase: whether visitor purchased
  • order_value: order value of visitor (in dollars)

Color

Instructions

  • map color to cyl
  • set color to 'blue'
# map color to cyl
ggplot(mtcars, aes(x = disp, y = mpg)) +
  geom_point(____)

# set color to 'blue'
ggplot(mtcars, aes(x = disp, y = mpg)) +
  geom_point(____)
# map color to cyl
ggplot(mtcars, aes(x = disp, y = mpg)) +
  geom_point(aes(color = cyl))

# set color to 'blue'
ggplot(mtcars, aes(x = disp, y = mpg)) +
  geom_point(color = 'blue')

Shape

Instructions
